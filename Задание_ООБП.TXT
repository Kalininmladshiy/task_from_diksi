Задание1.
	Существует бизнес-процесс, целью которого является ежемесячная отправка документов Контрагентам.
	Порядок выполнения:
	1. Сотрудник в определенную дату, один раз в месяц, выгружает документы в формате Excel из ERP-системы, тратя на это 20 часов суммарно в месяц (* далее везде суммарно)
	   - При работе в ERP-системе сотруднику виден исходный список всех документов, в количестве ~20 000, по всем Контрагентам (~5 000) на отчетную дату (исходные комплекты для всех одинаковые в системе: счет и акт)
       	   - По каждому Контрагенту необходимо выгрузить определенный комплект документов, который может отличаться по разным Контрагентам (только акт; либо только счет; либо оба документа сразу. Потребность задана в отдельном мастер-файле)
	   - Для выгрузки сотрудник фильтруется по каждому Контрагенту, фильтруется по типу документа, а далее сохраняет каждый документ в определенную папку
       - Итого, согласно мастер-файла, сотруднику необходимо выгрузить 2 400 файлов с учетом фильтров выше 

	2. После выгрузки документов сотрудник проводит их оформление: распечатывает, проставляет печать и подпись. На эти операции тратится 10 часов
	3. После этапа оформления сотрудник сканирует документы для отправки их по электронной почте, тратя на это 5 часов
	4. После сканирования сотрудник осуществляет рассылку документов по E-mail для каждого Контрагента, тратя на это 10 часов

	1 час работы сотрудника оценивается в 180 руб.

	Сформируйте в письменной форме (можно использовать блок-схемы) возможные на Ваш взгяд предложения по оптимизации данного бизнес-процесса (1-3 предложения-сценария), 
		предоставьте сравнительную таблицу с расчетом эффективности для всех вариантов бизнес-процесса,
		сформулируйте вывод о целесообразности оптимизации бизнес-процесса
	Оформите результат анализа не более, чем на 7 слайдах презентации, шрифт 18пт

Задание2.
	На любом из языков Python/C#/VBA создайте 2 программных кода (отдельные скрипты):
	1)  для формирования пустых файлов в формате *.txt с названием по шаблону "КА_<Код КА>_<тип_документа>_<дата>" по списку во вложении
	   <Код КА> - значение из столбца А
	   
	   <тип_документа> - значение из столбца B. 
	   !Значения типа документа могут быть указаны через запятую, в таком случае по каждому из них нужно сформировать отдельный файл
	   
	   <дата> - значение из столбца C
	   
	   Сохраните эти файлы в отдельную папку

	2) Получите список сохраненных файлов из папки и распределите их по подпапкам при выполнении условия:
	   Папка1) Файлы, по которым в мастер-файле в столбце D установлено значение 1
	   Папка2) Файлы, у которых в названии файла 2 последние цифры <Кода_КА> одинаковые, например, PA_00011, PA_00022
	   Папка3) Файлы, с датой в промежутке от 20.06.2020 до 10.07.2020
		   Если файл подходит под несколько условий, то необходимо делать его копию (копировать во все папки по условиям)
	Результат выполнения необходимо предоставить в виде кода программы, сохраненным в текстовом файле
